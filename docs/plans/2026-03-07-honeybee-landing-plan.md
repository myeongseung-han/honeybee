# 모래재꿀벌 랜딩 페이지 구현 계획

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 양봉업체 "모래재꿀벌" 소개 랜딩 페이지를 Next.js 16으로 구현하고 Vercel에 배포한다.

**Architecture:** Next.js 16 App Router 기반 싱글 페이지 랜딩. 5개 섹션(히어로, 브랜드 스토리, 제품, 갤러리, 연락처)으로 구성. Framer Motion으로 히어로 파티클 애니메이션 및 스크롤 등장 효과 구현.

**Tech Stack:** Next.js 16, React 19, TypeScript, Tailwind CSS v4, Framer Motion 12
